About The Project

What is RevLock?

RevLock (RevPasswordManager P2) is a secure, web-based password vault built as a full-stack Java project. It allows authenticated users to store, manage, and retrieve their credentials for various websites and services — all encrypted at rest.

The application is powered by Spring Boot on the backend with Spring Security handling authentication and session management. Credentials are encrypted using AES-256 before being persisted to a relational database via Spring Data JPA.

The frontend uses Thymeleaf server-side templating with a clean, responsive design system built with modern CSS and smooth micro-animations.

Features

Everything You Need to Stay Secure

RevLock packs a comprehensive set of security and usability features — engineered for developers and everyday users alike.

🔐

AES-256 Encryption

Every password stored in your vault is encrypted using AES-256-GCM before it ever touches the database. Your master credentials are never stored in plain text.

🛡️

Spring Security Auth

Session-based authentication with Spring Security. Passwords are hashed with BCrypt and brute-force protection keeps your account safe from automated attacks.

🎲

Password Generator

Generate cryptographically strong passwords with customisable length, uppercase, lowercase, numbers, and special character options in one click.

🗄️

Vault Management

Organise all your credentials in a clean, searchable vault. Add, edit, copy, or delete entries for websites, apps, and services at any time.

👤

User Profiles

Manage your account details, update your master password, and control your security preferences from a personalised profile dashboard.

✨

Modern Responsive UI

A polished, fully responsive interface with smooth animations and a clean design system — optimised for a great experience on desktop, tablet, and mobile.
